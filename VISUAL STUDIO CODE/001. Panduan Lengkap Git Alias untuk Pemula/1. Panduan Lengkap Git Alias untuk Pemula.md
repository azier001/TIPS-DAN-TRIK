## â“ FAQ (Pertanyaan yang Sering Ditanyakan)

### Q: Apakah alias bersifat permanen?
**A:** Ya! Alias yang dibuat dengan `--global` akan tersimpan permanen sampai kamu hapus.

### Q: Bagaimana cara menghapus alias?
**A:** Gunakan perintah:
```bash
git config --global --unset alias.<nama_alias>
```

### Q: Bisakah membuat alias untuk perintah yang kompleks?
**A:** Bisa! Contoh:
```bash
git config --global alias.last "log -1 HEAD"
```

### Q: Apakah alias mempengaruhi performa Git?
**A:** Tidak sama sekali! Alias hanya shortcut, tidak mengubah cara kerja Git.
