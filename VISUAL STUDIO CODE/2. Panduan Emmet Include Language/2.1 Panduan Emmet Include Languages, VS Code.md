## 🤔 Apa itu Emmet Include Languages?

**Emmet Include Languages** adalah fitur di VS Code yang memungkinkan kamu menggunakan shortcut HTML Emmet di file bahasa pemrograman lain seperti JavaScript, PHP, atau Vue.js.

### 🌟 Manfaat:
- 🚀 **Coding lebih cepat** dengan HTML shortcuts
- 🎨 **Konsisten** di berbagai jenis file
- 💪 **Produktivitas meningkat** untuk web developer
- 🔧 **Fleksibel** sesuai kebutuhan

### 📖 Contoh Sederhana:
```javascript
// Ketik: div.container>h1{Judul}+p{Konten}
// Tekan Tab
// Hasil: <div class="container"><h1>Judul</h1><p>Konten</p></div>
```

---

## ⚡ Setup Cepat

### 🎯 Cara Tercepat (Recommended):
1. Buka **Settings** (`Ctrl+,`)
2. Cari **"emmet include"**
3. Klik **"Add Item"** 
4. Tambahkan bahasa yang diinginkan

### 🔧 Cara Manual (settings.json):
```json
{
    "emmet.includeLanguages": {
        "javascript": "html",
        "php": "html",
        "vue": "html"
    }
}
```

## 🔧 Cara Pengaturan

### 🖥️ Metode 1: Menggunakan Settings UI

<details>
<summary>👆 Klik untuk melihat langkah lengkap</summary>

**Langkah 1: Buka Settings**
- Tekan `Ctrl+,` (Windows/Linux) atau `Cmd+,` (Mac)
- Atau buka `File → Preferences → Settings`

**Langkah 2: Cari Emmet**
- Ketik `emmet include` di kotak pencarian
- Temukan **"Emmet: Include Languages"**

**Langkah 3: Tambah Bahasa**
- Klik tombol **"Add Item"**
- Isi:
  - **Item**: `javascript` (bahasa sumber)
  - **Value**: `html` (mode Emmet)

**Langkah 4: Simpan**
- Settings otomatis tersimpan
- Restart VS Code jika diperlukan

</details>

### 📄 Metode 2: Edit settings.json

<details>
<summary>👆 Klik untuk melihat konfigurasi JSON</summary>

**Konfigurasi Lengkap:**
```json
{
    "emmet.includeLanguages": {
        "javascript": "html",
        "javascriptreact": "html",
        "typescript": "html",
        "typescriptreact": "html",
        "php": "html",
        "vue": "html",
        "twig": "html",
        "erb": "html"
    },
    "emmet.triggerExpansionOnTab": true,
    "emmet.showExpandedAbbreviation": "always"
}
```

**Cara Akses settings.json:**
1. Tekan `Ctrl+Shift+P`
2. Ketik `Preferences: Open Settings (JSON)`
3. Tambahkan konfigurasi di atas

</details>

### 🎯 Metode 3: Per File

Tambahkan komentar di awal file:
```javascript
// emmet:html
```

## 💡 Contoh Penggunaan

### 🚫 Sebelum Setup
```javascript
// Harus ketik manual - lama dan ribet
const element = document.createElement('div');
element.className = 'container';
element.innerHTML = '<h1>Judul</h1><p>Konten</p>';
```

### ✅ Setelah Setup
```javascript
// Ketik: div.container>h1{Judul}+p{Konten}
// Tekan Tab - langsung jadi!
<div class="container"><h1>Judul</h1><p>Konten</p></div>
```

### 🎯 Shortcut Emmet yang Sering Dipakai

| 💻 Ketik Ini | 🎉 Jadi Ini |
|-------------|-------------|
| `div.container` | `<div class="container"></div>` |
| `ul>li*3` | `<ul><li></li><li></li><li></li></ul>` |
| `h1+p` | `<h1></h1><p></p>` |
| `div#header.main` | `<div id="header" class="main"></div>` |
| `input:text` | `<input type="text">` |
| `a[href="#"]` | `<a href="#"></a>` |
| `img[src alt]` | `<img src="" alt="">` |
| `table>tr*2>td*3` | Tabel 2 baris 3 kolom |

## 🎯 Tips & Trik

### ⚡ Pengaturan Tambahan untuk Pengalaman Optimal

```json
{
    "emmet.includeLanguages": {
        "javascript": "html",
        "vue": "html"
    },
    "emmet.triggerExpansionOnTab": true,
    "emmet.showExpandedAbbreviation": "always",
    "emmet.showSuggestionsAsSnippets": true,
    "emmet.variables": {
        "lang": "id",
        "charset": "UTF-8"
    }
}
```

### 🚀 Shortcut Emmet Advanced

<details>
<summary>👆 Klik untuk melihat shortcut tingkat lanjut</summary>

**Struktur Kompleks:**
```
div.wrapper>header.header>nav.nav>ul>li*4>a[href="#"]{Menu $}
```

**Form Lengkap:**
```
form[method="POST"]>div.form-group*3>label+input:text
```

**Card Layout:**
```
div.card>div.card-header>h3{Judul}^^div.card-body>p{Konten}
```

**Grid System:**
```
div.row>div.col-md-4*3>div.card>div.card-body>h5{Card $}+p{Konten card}
```

</details>

## 🔍 Pemecahan Masalah

### 🚨 Masalah Umum & Solusi

<details>
<summary>❌ Emmet tidak bekerja setelah setup</summary>

**Solusi:**
- ✅ Restart VS Code
- ✅ Cek apakah bahasa file terdeteksi dengan benar
- ✅ Pastikan settings tersimpan dengan benar
- ✅ Coba ketik ulang konfigurasi

</details>

<details>
<summary>❌ Tab tidak meng-expand abbreviation</summary>

**Solusi:**
- ✅ Aktifkan `"emmet.triggerExpansionOnTab": true`
- ✅ Cek apakah ada extension yang konflik
- ✅ Pastikan cursor di akhir abbreviation
- ✅ Coba tekan `Ctrl+Space` untuk melihat suggestions

</details>

<details>
<summary>❌ Tidak muncul suggestions</summary>

**Solusi:**
- ✅ Set `"emmet.showExpandedAbbreviation": "always"`
- ✅ Aktifkan `"emmet.showSuggestionsAsSnippets": true`
- ✅ Coba ketik beberapa karakter dulu baru tekan `Ctrl+Space`

</details>

<details>
<summary>❌ Emmet bekerja tapi hasil tidak sesuai</summary>

**Solusi:**
- ✅ Cek mapping bahasa apakah sudah benar
- ✅ Pastikan syntax Emmet yang digunakan benar
- ✅ Coba gunakan mode `css` untuk CSS abbreviations
- ✅ Baca dokumentasi Emmet untuk syntax yang tepat

</details>
