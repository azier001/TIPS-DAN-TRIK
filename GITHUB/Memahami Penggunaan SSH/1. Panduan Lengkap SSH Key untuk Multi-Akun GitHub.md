## 5ï¸âƒ£ Konfigurasi Multi-Akun

### âœï¸ Isi File Config

```
    HostName github.com
    User git
```

### ğŸ’¡ Penjelasan Config

|--------|------------|

| `User git` | Username yang digunakan (selalu `git` untuk GitHub) |
| `IdentityFile` | Path ke private key yang akan digunakan |
| `IdentitiesOnly yes` | Paksa gunakan key ini saja, jangan coba key lain |

**ğŸ¯ Konsep Penting:**
- `github.com-andi0027` adalah **alias** yang Anda buat sendiri
- Saat Anda pakai alias ini, SSH otomatis pakai key `id_ed25519_andi0027`

## 9ï¸âƒ£ Troubleshooting

### âŒ Error: "Permission denied (publickey)"

**Penyebab:**
- Public key belum diupload ke GitHub
- Menggunakan key yang salah
- Permission file tidak benar

**Solusi:**

**1. Cek apakah key sudah diupload:**
- Buka https://github.com/settings/keys
- Login dengan akun yang bermasalah
- Pastikan key ada di list

**2. Test koneksi dengan verbose:**
```bash
ssh -vT git@github.com-ahmad027
```

Output akan tampilkan detail koneksi. Cari bagian:
```
debug1: Offering public key: /Users/azier/.ssh/id_ed25519_ahmad027
```

**3. Set permission (Mac/Linux):**
```bash
cd ~/.ssh
chmod 600 id_ed25519_*
chmod 644 id_ed25519_*.pub
```

---

### âŒ Error: "Could not resolve hostname github.com-ahmad027"

**Penyebab:**
- File config salah/tidak ada
- Typo di nama alias

**Solusi:**

**1. Cek isi config:**
```bash
cat ~/.ssh/config
```

**2. Pastikan ada entry untuk alias tersebut:**
```

    HostName github.com
    ...
```

**3. Cek typo:**
- âœ… Benar: `github.com-ahmad027`
- âŒ Salah: `github.com ahmad027` (pakai spasi)
- âŒ Salah: `github-ahmad027` (kurang `.com`)

---

### âŒ Error: Commit tercatat dengan nama yang salah

**Penyebab:**
- Git config tidak di-set per repository
- Masih pakai global config

**Solusi:**

**1. Cek config aktif:**
```bash

git config user.name
git config user.email
```

**2. Set ulang config yang benar:**
```bash
git config user.name "ahmad027"
git config user.email "mutaahmad10@gmail.com"
```

**3. Verifikasi:**
```bash
git config --local --list
```

Harus ada:
```
user.name=ahmad027
user.email=mutaahmad10@gmail.com
```

---

### âŒ Error: "fatal: Could not read from remote repository"

**Penyebab:**
- Remote URL salah
- Tidak punya akses ke repository

**Solusi:**

**1. Cek remote URL:**
```bash

```

**2. Pastikan format benar:**
```

```

Perhatikan:
- âœ… Pakai alias: `github.com-ahmad027`
- âœ… Username benar: `ahmad027`

**3. Ubah jika salah:**
```bash

```

---

### âŒ SSH Agent Issues

Kadang SSH agent tidak otomatis load key Anda.

**1. Start SSH agent:**


**2. Add key ke agent:**
